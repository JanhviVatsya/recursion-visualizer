# Recursion Visualizer

A backend application built with Spring Boot that visualizes recursion by executing user-defined recursive functions inside a secure Docker sandbox.

## Tech Stack

- Java 21
- Spring Boot
- Docker
- Maven

## Planned Features

- Execute recursive C++ functions safely
- Generate recursion trees
- Visualize function calls
- Docker sandboxing
- Backtracking visualization (N-Queens, Sudoku, etc.)
